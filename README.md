# Huippumyyjän Päivärutiini

Valmis Next.js-projekti Verceliä varten.

## GitHubiin

Vie nämä tiedostot suoraan repon juureen:

- package.json
- next.config.js
- app/layout.jsx
- app/page.jsx
- app/styles.css
- README.md
- .gitignore

Tärkeää: `package.json` pitää näkyä GitHub-repon etusivulla, ei alikansion sisällä.

## Vercel

- Framework Preset: Next.js
- Build Command: `next build`
- Output Directory: jätä tyhjäksi
