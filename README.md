# Huippumyyjän Päivärutiini Pro

Ammattimainen Next.js-sovellus myynnillisen itsevarmuuden, vakuuttavuuden ja klousausrohkeuden päivittäiseen harjoitteluun.

## Sisältö

- Päivän myyntikompassi
- 30 päivän identiteettilause
- Aamurutiini 15 min
- 90 sekunnin pre-call-protokolla
- Klousauslauseet
- Päivän jälkeinen voittopankki
- Selaimen localStorage-tallennus

## GitHubiin

Vie nämä suoraan repon juureen:

- package.json
- next.config.js
- app/layout.jsx
- app/page.jsx
- app/styles.css
- README.md
- .gitignore

Tärkeää: `package.json` pitää näkyä GitHub-repon etusivulla.

## Vercel

- Framework Preset: Next.js
- Build Command: `next build`
- Output Directory: jätä tyhjäksi
